Отметьте здесь те рубрики, к которым должен иметь доступ выбранный менеджер по продажам.

При выборе рубрики, имеющей подрубрики автоматически отмечаются все её подрубрики. При этом, если вы снимете флажок с любой подрубрики, к ней менеджер иметь доступ не будет.

Если будет включена подрубрика неотмеченной рубрики, менеджер будет иметь доступ только к включенной подрубрике.

В доступные рубрики конечного уровня менеджер по продажам сможет добавлять новые товары (услуги).

Менеджеру будет доступен интерфейс управления характеристиками доступных ему рубрик.

Также менеджер сможет добавлять новые подрубрики в доступные, если их глубина не ограничена глобальными переменными (раздел "Конфигурация"). Добавленные менеджером рубрики по умолчанию будут доступны только ему.
