Интерфейс управления шаблонами новостей.

Поля "Заголовок" и "Текст" обязательны для заполнения.

Для всех полей доступны переменные (например "Наименованеи товара", или "Дата добавления"). Для каждого события эти переменные могут различаться. Для текущего события все переменные перечислены слева от поля "Текст" коричневым цветом, и рядом их описание.
Этими переменными можно пользоваться:

  • Нажав на неё. В этом случае переменная окажется в конце поля "Текст".
  • Выделить мышью, нажать правой кнопкой мыши на выделение, выбрать пункт "Копировать", затем указать мышью на любое из текстовых полей, нажать правой кнопкой мыши, в появившемся меню нажать "Вставить".

Отдельно стоит описать 2 переменные "{url}" и "{/url}".
Эти переменные служат для вставки в новость ссылки на текущее событие (например при добавлении товара ссылка будет ссылаться на страницу этого товара). Начало ссылки отмечается переменной "{url}", а конец "{/url}", при том между ними обязательно должен быть любой другой текст (в т.ч. могут быть и другие переменые). Если текста там не будет, ссылка проигнорируется.

При будующих генерациях новостей по шаблонам, переменные автоматически будут заменены на нужное значение.

Поле "Доступен к показу" означает, будет ли этот шаблон использоваться для генерации новостей.

Для сохранения шаблона нажмите кнопку "Отправить". Для отмены создания (отмены изменений) нажмите кнопку "Отмена".
Помните, что шаблон всегда можно изменить.
