Для добавления характеристики перейдите по ссылке "добавить" в верхней части страницы.

В основной части страницы представлена таблица характеристик.
В первом столбце перечислены наименования характеристик.
В последнем столбце "Действия" доступны следующие функции:
  • Справочник - управление списком возможных вариантов значений характеристики.
  • Ред. - изменение наименования и опций характеристики.
  • Вкл./Выкл. - включение/выключение характеристики.
  • Удалить - удаление характеристики без возможности восстановления.
