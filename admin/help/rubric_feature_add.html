Поля для ввода:
  1. Наименование - наименование характеристики. Поле обязательно к заполнению. Максимальная длина 255 символов.
  2. Обязательна к заполнению - если отмечено, то товар нельзя будет добавить без указания этой характеристики.
  3. Тип данных - это тип данных, которые нужно будет ввести при добавлении товара
    • Число - Допустимы к вводу все цифры, и знаки "точка" (.), "запятая" (,).
    • Текст - Для каждого товара будет доступно поле для ввода текста. Максимальное кол-во симолов - 255.
    • Логический - Возможны ответы "да" или "нет".
    • Справочник - Сначала вводятся все возможные значения характеристики, а при добавлении товара осуществляется выбор одного из этих значений.
  4. Включена - Будет ли доступна эта характеристика.
