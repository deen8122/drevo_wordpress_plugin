Для добавления аккаунта перейдите по ссылке "добавить" в верхней части страницы.

В основной части страницы представлена таблица аккаунтов.
В первом столбце перечислены "логины", по которым происходит авторизация. После логина показан адрес электронной почты, по которому вы сможете связаться с владельцем аккаунта.
Во втором столбце перечислены ФИО владельцев.
В последнем столбце "Действия" доступны следующие функции:
  • Привязать к рубрикам (доступно для менеджеров по продажам) - управление связями "Рубрикатор-Менеджер по продажам". Другими словами управление доступом конкретного менеджера к рубрикам.
  • Пароль - изменение пароля на доступ к аккаунту.
  • Ред. - изменение логина, e-mail, информации о владельце.
  • Вкл./Выкл. - отключение аккаунта. Владелец не сможет войти под текущим аккаунтом, однако аккаунт будет существовать, и в любое время может быть восстановлен.
  • Удалить - удаление аккаунта без возможности восстановления.
